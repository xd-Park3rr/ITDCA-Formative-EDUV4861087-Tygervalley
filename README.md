# ITDCA Formative Assessment - Data Structures

This repository contains implementations for the ITDCA formative assessment covering AVL Trees, Priority Queues, and Sorting Algorithms.

---

## Question 1 & 2: AVL Tree

**Location:** `AVLTree/`

### Question 1
Basic AVL tree implementation with:
- Insert and Delete operations with automatic balancing
- In-order traversal
- Rotations (LL, RR, LR, RL)

### Question 2
Enhanced AVL tree (reuses Q1 code) with additional features:
- In-order traversal showing titles only
- Binary search by year
- Display most recent book
- Display total book count

**Run:**
```bash
cd AVLTree
dotnet run
```

---

## Question 3: Priority Queue

**Location:** `PriorityQueue/`

Priority queue for patient management with priorities 0 (most urgent) to 4 (least urgent).

**Commands:**
- `ENQUEUE <name> <priority>`
- `DEQUEUE`

**Run:**
```bash
cd PriorityQueue
dotnet run
```
