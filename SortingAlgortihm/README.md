# Sorting Algorithm Project

## Files

**Program.cs** - Main program that loads hotel data and sorts by different metrics

**q4_QuickSorter.cs** - QuickSort implementation with support for multiple sorting criteria

**q4_JsonLoader.cs** - Loads hotel data from JSON file

**Properties/q4_Hotel.cs** - Hotel data structure

**data/hotels.json** - Sample hotel data

## Run

```
dotnet run
```
