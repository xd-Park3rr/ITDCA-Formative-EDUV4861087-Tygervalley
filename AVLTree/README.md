# AVL Tree Project

## Files

**Program.cs** - Main program that runs demos for Q1 and Q2

**q1_AVLTree.cs** - Basic AVL tree with insert, delete, and in-order traversal

**q2_AVLTree.cs** - Enhanced AVL tree that wraps q1 and adds extra features (count, search, etc.)

**properties/q1_AVLNode.cs** - Node structure for q1 AVL tree

**properties/q1_Book.cs** - Book class used in q1 tree

**properties/q2_AVLNode.cs** - Node structure for q2 AVL tree  

**properties/q2_Book.cs** - Book class used in q2 tree

## Run

```
dotnet run
```
